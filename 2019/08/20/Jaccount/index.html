发表于 2019-08-20

众所周知，国际大厂经常对在校大学生给予各种各样的优惠，从网购优惠到免费授权正版软件、开发工具。以买下Adobe全家桶远近闻名的SJTU自然在其它方面也加带了力度。

开门见山，今天以白嫖大厂享受高校资源为目的，为读者梳理一下怎么用Jaccount赚回学费获得更多学习资源。

本文将会只注重于介绍各种可嫖的资源，各优惠具体申请方式请善用搜索引擎。

$500一年

  • CLion - 跨平台的C/C++ IDE 开发工具，支持C11 、C14、libc++以及Boost。
  • IntelliJ IDEA - 2001年发布。一套智能的 Java 集成开发环境，特别专注与强调程序师的开发撰写效率提升。
  • PyCharm - 一款结合了Django框架的Python IDE开发工具。
  • WebStorm - JavaScript的开发工具。

想尝试学习开发的同学可以搞快点。

Amazon AWS

AWS是由亚马逊公司所创建的云计算平台，提供许多远程Web服务。

用Jaccount注册能在AWS上每年嫖100刀教育优惠（需要信用卡。这里实名建议尽早整一张信用卡，反正迟早要整的）。要知道Lightsail的最低配主机只需要3.5刀每月……

然后这个Amazon Lightsail真的实力造福人类。Lightsail可创建实例内置了从Wordpress到LAMP各种各样的配置好的应用，主机区域包括俄亥俄州、弗吉尼亚州、东京、首尔等~~（疯狂暗示）~~。

腾讯云

AWS能白嫖的主机毕竟在国外，如果你的应用对延迟有要求~~（再次迷惑暗示）~~会很尴尬。

腾讯云推出10元每月的校园优惠套餐（配置是单核CPU、2G内存，1Mbps带宽和50G存储），加16元可选购1年.cn域名使用权。

当然还有阿里云和百度智能云等可供选择，大家按各自喜好选择厂家……
